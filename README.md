📊 Sales Analytics Dashboard

📌 Project Overview

This project analyzes retail sales data and provides insights using Python, SQL, and Streamlit.

🛠️ Tools Used

- Python
- Pandas
- SQLite (SQL)
- Streamlit

📊 Features

- Interactive dashboard with filters
- Sales trend visualization
- Monthly sales analysis
- Top performing stores
- SQL-based data analysis

🧠 Key Insights

- Top stores contribute majority of total revenue
- Sales show seasonal trends across months
- Some stores consistently outperform others
- Holiday periods influence sales patterns

🚀 How to Run

pip install -r requirements.txt
streamlit run app.py
